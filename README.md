AI Tic Tac Toe 
This is a web-based Tic Tac Toe game built with **Python, Flask, and JavaScript.  
The game allows a player to compete against an **AI opponent** with two difficulty levels.
The project started as a simple command-line Tic Tac Toe game and was later upgraded into a full web application with an AI opponent and interactive UI.

Features
- Play Tic Tac Toe against an AI
- Two difficulty levels:
  - Easy (random AI moves)
  - Impossible (Minimax AI that never loses)
- Interactive game board
- Scoreboard to track wins
- Winner screen when the game ends
- Sound effects for moves, wins, and draws
- Winning cell animation
- Responsive design (works on mobile and desktop)


Technologies Used
- **Python**
- **Flask**
- **JavaScript**
- **HTML**
- **CSS**
- **Minimax Algorithm (AI)

How the AI Works
The AI uses the **Minimax algorithm**, which evaluates all possible future moves and selects the best one.
This makes the AI unbeatable when the **"Impossible" difficulty** is selected.
